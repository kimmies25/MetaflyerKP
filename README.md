# Code convention
1. Class
   1. Class names use Pascal Case techniques
2. Method
   1. Method names use Snake Case techniques
3. Variable
   1. Variable names use Snake Case techniques
4. Constants
   1. Constant names use Upper Case techniques (example: DEFAULT_SLEEP)
